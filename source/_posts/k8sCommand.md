1. watch 状态
```
kubectl get pods --watch
```

2. kubectl get 资源名  -> 简单描述

```
kubectl get rc
kubectl get pods
kubectl get deployment
```


3. kubectl describe 资源名  -> 详细信息

```
kubectl describe rc
kubectl describe pods
kubectl describe deployment
```

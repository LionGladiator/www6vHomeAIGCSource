### stage == index 


1. git提交到远程分支
```
git add .
git commit -m  '相关注释文字'
git push origin master
```

2. 一个分支（master）代码同步到另一个分支（dev）
```
git stash 
git fetch origin master 
git rebase origin dev 
git stash pop
```

3. 新建分支
```
git checkout -b dev
```

4. 强制覆盖本地文件
```
git fetch --all
git reset --hard origin/master
git reset --hard origin/<branch_name>
```
